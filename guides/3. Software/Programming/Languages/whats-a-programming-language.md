# What's a Programming Language?
A [programming language](https://en.wikipedia.org/wiki/Programming_language) is how computer programs are expressed and written in a human-readable way.

Each programming language has its own [syntax](https://en.wikipedia.org/wiki/Syntax_(programming_languages)), or 'grammar' which define the rules of how keywords can be used.

## Keywords
Programming languages usually consist of human-readable [keywords](https://en.wikipedia.org/wiki/Reserved_word).

Common keywords include (in pseudocode):
- if *(condition)*, then: *(code to execute)*, else: *(code to execute)*
- while *(condition)*, do: *(code to execute)*
- for *(iterator variable name)* in *(existing iterable variable)*, do: *(code to execute)*

# Graphics Libraries & APIs

(w) 'Graphics Library' is often used interchangeably with 'Graphics API'. This guide uses these two terms differently.

## Graphics APIs
A [Graphics API](graphics-api) is a type of [API](whats-an-api) which allows access to a computer's graphics hardware for rendering 2D and/or 3D graphics.

Graphics APIs are also known as 'Rendering APIs'.

## Graphics Libraries
A [Graphics Library](graphics-library) is a [library](library) which abstracts away the nitty-gritty details of a Graphics API, and provider simpler interfaces for developers to have their code use.

Developing using a graphics library is often easier than with a graphics API, but often comes at the cost of performance and flexibility.

Some packages bundle in graphics libraries along with sound libraries, physics engines and more (See [Game Engine](game-engine)).

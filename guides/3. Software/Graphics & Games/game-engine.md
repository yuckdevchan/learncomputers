# Game Engine
A [Game Engine](game-engine) is a [software application](application) or [library](library) that **give developers** a **foundation** to make **video games**.

Using a **game engine** can **save time** and **effort** for developers, as they **don’t** have to build **everything from scratch**.

Some **game engines** are for **3D**, **2D**, and some are **for both**.

Some **game engines** including [Unity](unity) include a [GUI](gui) **editor application**, others are a [code library](library) **only**.

## Popular game engines
- [Unity](unity) - **2D** & **3D** **game engine** with a **GUI editor** and **scripting support**.
- [Unreal Engine](unreal-engine) - **2D** & **3D** **game engine** with a **GUI editor** and **scripting support**.
- [Godot](godot) - **2D** & **3D** **game engine** with a **GUI editor** and **scripting support**.

# Linux
[Linux](https://en.wikipedia.org/wiki/Linux) is a family of [operating systems](/guides/software/operating-systems/whats-an-operating-system) which are built around the [Linux](https://en.wikipedia.org/wiki/Linux_kernel) [kernel](/guides/software/operating-systems/components-of-an-os/kernels).

There is no single operating system called 'Linux'. Instead, there are many [distributions](https://en.wikipedia.org/wiki/Linux_distribution), or 'distros'.

Almost all Linux distros use the GNU core utilities, which is why the term 'GNU/Linux' is often used instead of just 'Linux'. See: [Linux naming controversy](https://en.wikipedia.org/wiki/GNU/Linux_naming_controversy).

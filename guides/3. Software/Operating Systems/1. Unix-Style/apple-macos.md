# Apple macOS
[macOS](https://en.wikipedia.org/wiki/MacOS) is an [operating system](whats-an-operating-system) developed by [Apple Inc.](https://en.wikipedia.org/wiki/Apple_Inc.)

(w) macOS is not to be confused with [Classic Mac OS](https://en.wikipedia.org/wiki/Classic_Mac_OS) which was its own separate operating system. macOS used to be called "Mac OS X and then "OS X" before being renamed to "macOS" in 2016.

macOS is based on [FreeBSD](bsd#freebsd) with components taken from the [NeXTSTEP](https://en.wikipedia.org/wiki/NeXTSTEP) operating system.

Because macOS and [Linux](linux) are both [unix-style](whats-unix) operating systems, software can be easily developed for both at the same time.

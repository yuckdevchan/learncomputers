# Text Editors
[Text editors](https://en.wikipedia.org/wiki/Text_editor) are applications used for manipulating textual documents.

## Programming
Special text editors are often used for programming.

Text editors for programming usually include features such as:
- [Syntax highlighting](https://en.wikipedia.org/wiki/Syntax_highlighting)
- Built-in LSP
- [Breakpoint](https://en.wikipedia.org/wiki/Breakpoint) management
- [Version control](https://en.wikipedia.org/wiki/Version_control)
- A [console pane](https://en.wikipedia.org/wiki/Terminal_emulator)

Text editors with many programming-focused features are often called IDEs ([Integrated Development Environments](https://en.wikipedia.org/wiki/Integrated_development_environment)). There is no definitive line between an IDE and a text editor with programming features.

Popular IDEs include:
- [Visual Studio Code](https://en.wikipedia.org/wiki/Visual_Studio_Code)
- [Visual Studio](https://en.wikipedia.org/wiki/Visual_Studio)
- [JetBrains PyCharm](https://en.wikipedia.org/wiki/PyCharm)
- [JetBrains IntelliJ IDEA](https://en.wikipedia.org/wiki/IntelliJ_IDEA)
- [Eclipse](https://en.wikipedia.org/wiki/Eclipse_(software))

Popular text editors with programming features include:
- [vi](https://en.wikipedia.org/wiki/Vi_(text_editor))
- [vim](https://en.wikipedia.org/wiki/Vim)
- [GNU Emacs](https://en.wikipedia.org/wiki/GNU_Emacs)
- [Sublime Text](https://en.wikipedia.org/wiki/Sublime_Text)

# NIC
[NIC](https://en.wikipedia.org/wiki/Network_interface_controller) stands for network interface controller.

(w) **NIC** can also stand for **network interface card**, which is a **detachable component** which **includes** a **network interface**.

A **NIC** is the **component** in a computer that allows it to **connect to a network**, including to the [internet](internet).

**NICs** can **support** wired [networking](whats-networking), [wireless networking](https://en.wikipedia.org/wiki/Wireless_network), or both.

**Wired NICs** almost always **include** an [ethernet](ethernet) **port**, so they can be plugged into a [network switch](network-switch).

**Wireless NICs** almost always **include** a [Wi-Fi](wi-fi) **antenna**, so they can **connect to** a **wireless access point**.

(e) One computer can have **multiple NICs**. Additional **NICs** can be connected via **USB**. Most **computers** with one **NIC** can still connect to **multiple networks** by using **wireless** and **wired** networking **simulatenously**.

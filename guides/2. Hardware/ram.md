# RAM
[RAM](https://en.wikipedia.org/wiki/Random-access_memory) stands for **Random Access Memory**

**RAM** is where a **computer** can **store data** it is **currently using**.

(e) **RAM** is like a computer's **short-term memory**.

When a **computer** is **turned off**, its **RAM is cleared** and all its **data is lost**.

**RAM** is **more expensive** than **storage**, but is **much faster**.

**RAM** is **also known** as [primary storage](Computer_data_storage#Primary), **main memory**, or [volatile memory](volatile-memory).

## Structure of data
Data in **RAM** is stored in [binary](binary) (0s and 1s).

Data is usually addressed in [bytes](byte) (8 bits). So, each memory address in **RAM** corresponds to a **byte of data**.

The address is what the [CPU](cpu) sends to **RAM** when it wants to **read** or **write** data in that address.

## Swap
If a **computer** doesn't have **enough RAM** to run a **program**, it can **allocate** a **portion** of **secondary storage** (usually [HDD](hdd) or [SSD](ssd)) as **virtual memory**. This is called **swap** because **the data** can be **swapped** between **RAM** and **secondary storage**.
